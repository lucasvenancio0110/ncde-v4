# NCDE V4

Motor de decisão para planejamento de revezamentos, jantares, ajustes e setups em operações CNC.

## Princípio operacional

1. Proteger setups futuros.
2. Proteger a produção.
3. Manter capacidade de reação.
4. Organizar revezamentos.
5. Distribuir jantares.

## Desenvolvimento

```bash
npm install
npm run dev
```

## Verificações

```bash
npm run test
npm run build
npm run lint
```

O projeto usa React, Vite, TypeScript em modo estrito e Vitest.
